Soluzione 2: Controllo della Dimensione Font e Padding
Se il problema persiste, controlla che i font e i padding non siano troppo grandi, causando overflow. Valuta di ottimizzare questi valori:

/* Esempio di CSS */
.header-title {
    font-size: 1.25rem; /* Riduci se necessario */
    padding: 0.5rem; /* Controlla i padding */
}
Soluzione 3: Gestione dei Media Queries
Utilizza media queries per gestire l’aspetto del layout su schermi più piccoli.

@media (max-width: 768px) {
    .header-title {
        font-size: 1rem; /* Modifica dimensione per schermi piccoli */
    }
}
Dopo aver applicato queste modifiche, verifica se la visualizzazione UX è migliorata. Se ci sono ancora problemi, potremmo dover esaminare altre parti del layout o il CSS.
